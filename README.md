
# Quiz App - React

A simple and interactive quiz application built with React to test your programming knowledge, specifically focused on JavaScript concepts.

## 🚀 Features

- Multiple choice questions about JavaScript
- Real-time score tracking
- Visual feedback for selected answers
- Results summary at the end of the quiz
- Reset functionality to retake the quiz
- Responsive design

## 📋 Prerequisites

Before you begin, ensure you have the following installed:

- [Node.js](https://nodejs.org/) (v14.0.0 or higher)
- [npm](https://www.npmjs.com/) (v6.0.0 or higher) or [yarn](https://yarnpkg.com/)
